# Add Practice Exercises Skill

Add three-tier practice exercises to a section.

## Arguments
- `$ARGUMENTS` - Section path (e.g., "M1/C3/S5")

## Instructions

Add exercises at three difficulty levels:

### Exercise Structure
```markdown
## Practice Exercises

### Exercise 1: Basic - [Title]
**Objective:** [What skill this practices]
**Time:** ~10 minutes

[Clear instructions]

```bash
# Commands or code to run
```

**Expected Result:** [What success looks like]

<details>
<summary>Hint (click to reveal)</summary>
[Helpful hint without giving away answer]
</details>

---

### Exercise 2: Intermediate - [Title]
**Objective:** [Combines multiple concepts]
**Time:** ~20 minutes

[Instructions requiring synthesis]

**Starter Code:**
```python
# [Partial code for student to complete]
```

**Success Criteria:**
- [ ] [Criterion 1]
- [ ] [Criterion 2]

---

### Exercise 3: Challenge - [Title]
**Objective:** [Extends beyond section content]
**Time:** ~30+ minutes

[Open-ended challenge]

**Hints:**
1. Consider [approach 1]
2. Research [related topic]

**Bonus:** [Extra challenge for advanced students]
```

### Guidelines
- Exercise 1: Direct application of section content
- Exercise 2: Requires combining 2-3 concepts
- Exercise 3: Open-ended, encourages research
- Include time estimates
- Provide hints without answers
- Connect to capstone where possible
