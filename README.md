# Physical AI & Humanoid Robotics Book

**Author**: Muhammad Junaid Sajjad
**Institution**: Panaversity

## Overview
Revolutionary digital textbook for Physical AI and Humanoid Robotics. Transforms traditional education with AI-powered interactive learning, RAG-based Q&A, and immersive multimedia content.

## Key Features
- **AI-Powered Learning**: RAG-based chatbot answers questions from book content
- **Interactive Experience**: Live VS Code simulation and multimedia content

## Technology Stack
- **Frontend**: Docusaurus v3.6, React, TypeScript
- **Backend**: FastAPI, Python, Langchain, Qdrant
- **AI Services**: RAG, Translation, Personalization
- **Simulation**: ROS 2, NVIDIA Isaac Sim, Gazebo

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm start`
4. Build for production: `npm run build`

## Modules
- **M1**: The Robotic Nervous System (ROS 2 Kilted Kaiju)
- **M2**: The Digital Twin Hallucination (Gazebo Simulation)

- **M4**: Vision-Language-Action Embodiment (VLA Integration)
